## Agent skills

- Context: in-repo — domain context and ADRs live in this repository; see `docs/agents/domain.md`.
- Issues: local markdown files under `.scratch/`; no triage labels; see `docs/agents/issue-tracker.md`.
- Layout: multi-context — `CONTEXT-MAP.md` at the repo root points to root and `.config` contexts.
- Commit style: use concise Conventional Commit-style subjects, matching repo history (for example, `chore: update configs`).
